# WEB POS LARAVEL (APLIKASI KASIR)
aplikasi point of sale (POS) berbasis laravel untuk mengelola produk,transaksi penjualan dan laporan.

#  FITUR 
- Manajemen Produk
- Transaksi Penjualan
- Perhitungan Total & Kembalian
- Laporan Penjualan
- Sistem Stok Barang

# teknologi 
- Laravel
- PHP
- MySQL
- Bootstrap


# CARA INSTALL
1. Clone repository
   git clone https://github.com/rimbirenata/web-pos-laraveel.git
2. Install dependency
   composer install
3. Copy env
   cp .env.example .env
4. Generate key
   php artisan key:generate
5. Migrasi database
   php artisan migrate
6. Jalankan server
   php artisan serve
